加入我们

这条路没有确定的回报，但有真实的机会。

你能得到什么？

🌱 短期（1-3个月）

  • 技能提升：通过协作学习
  • 认知升级：AI辅助理解复杂问题
  • 真实连接：找到同频的人

🌿 中期（3-6个月）

  • 从被动到主动：自己找事做
  • 可见的进步：贡献记录上链
  • 协作网络：与他人深度协作

🌳 长期（6个月+）

  • 非线性成长：三条腿相乘的机会
  • 身份认同：不可篡改的贡献历史
  • 参与更大的事：从个人到共同体

我们需要什么？

✅ 我们想要的人

  • 接受不确定性：这里没有确定的回报
  • 主动努力：不等待指令，自己找事做
  • 真实反馈：从底层视角提供真实问题

❌ 我们不想要的人

  • 寻求确定性：需要稳定承诺的
  • 被动等待：等别人分配任务的
  • 道德绑架：用情怀绑架他人的

如何开始？

  1. 注册 GitHub 账号（如果还没有）：github.com/signup
  2. 提交申请：点击下方按钮，填写申请表单
  3. 我们会在 2-3 天内审核你的申请
  4. 审核通过后，系统自动将你加入人才库
  5. 开始认领任务，参与协作

🚀 一键申请

点击下方按钮，在 GitHub 上提交你的申请。填写表单即可，无需懂 Git。
